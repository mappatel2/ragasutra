# Ragasutra 🎵🕰️

**Ragasutra** is a Unity-based application designed to harmonize the listener's environment with the time of day through Indian Classical Music.

## 📖 Overview
In Indian Classical tradition, specific Ragas are meant to be performed during specific time cycles called **Prahars**. Ragasutra automates this connection. The app detects the device's system time and plays the corresponding Raga (e.g., *Bhairav* in the morning, *Yaman* in the evening), creating an immersive audio-visual backdrop for meditation, work, or relaxation.

## ✨ Key Features
* **Prahar Logic:** Automated time-detection mapping 24 hours to the 8 traditional musical periods.
* **Dynamic Visuals:** Unity-powered backgrounds that shift lighting and ambience from dawn to dusk.
* **Cross-Platform:** Built for potential deployment on Android, iOS, and WebGL.
